## Purpose

These files are kept for:
1. **Reference** - Example implementations and patterns
2. **Code reuse** - Copy components or logic when needed
3. **UI inspiration** - Design patterns and layouts
4. **Template features** - Access to original VRISTO features

## Note

**Do not delete this directory.** The template files may be useful for:
- Implementing new features with similar UI patterns
- Learning React/TypeScript patterns used in the template
- Quick prototyping of new pages
- Understanding the original template structure

If you need any of these features in your application, you can:
1. Copy the relevant file(s) to the appropriate location in `/src/pages`
2. Update imports and adapt to your data models
3. Add routing in `/src/router/routes.tsx`
4. Update the main pages README
